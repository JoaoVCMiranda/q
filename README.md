# Primeiros passos

## Criar um ambiente virtual

Para evitar o risco de conflito com outros pacotes já instalados no sistema 

```sh 
python -m venv .venv
```

### Entrar no ambiente virtual também

```sh 
source .venv/bin/activate
```



## Instalar os pacotes

Instalação via pip


```sh 
pip install ket-lang
```
